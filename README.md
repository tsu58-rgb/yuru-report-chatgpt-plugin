# ゆる歴史散歩 ChatGPT レポート投稿

ゆる歴史散歩会のWordPressへ、カスタムGPTから散歩イベントの開催レポートを下書き投稿する専用プラグインです。

この公開リポジトリは、WordPressプラグインの自動更新元として使用します。サイト固有のAPIキーやWordPressの認証情報は含みません。

## 現在のバージョン

1.4.0

## 主な機能

- 散歩レポートの下書き投稿
- 画像のWebP化と英数字ファイル名
- 投稿者・カテゴリーの固定
- Cocoon向けSEOタイトルとメタディスクリプション
- メディアライブラリ内の行政地図を再利用
- GitHub経由の自動更新
